# Programs

This repository contains a collection of programs developed by gregpuzzles1. These programs were designed to run specifically on Raspberry Pi devices, including the Raspberry Pi 5 and Raspberry Pi Zero 2 W.

## About

The main purpose of this repository is to house various utility scripts and applications for Raspberry Pi platforms. Each program serves a unique function, ranging from system management to custom automation tasks.

## Supported Devices

- **Raspberry Pi 5**
- **Raspberry Pi Zero 2 W**

Feel free to explore, modify, and use these programs to enhance your Raspberry Pi experience!

## Getting Started

To use these programs:
1. Clone the repository:
    ```bash
    git clone https://github.com/gregpuzzles1/Programs.git
    ```
2. Follow the instructions in each program's source file or accompanying documentation for specific dependencies and usage steps.
